HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.

CSS

  • CSS Rules : selector {property: value;}
  • A margin indicates how much space we want around the outside of an 9

Git

  • git status: Checks what branch we are currently working on.
  • git checkout -b branch-name : creates a new branch and switches to it.
  • git checkout branch-name : switches to the named branch
  • git add -A : command to commit all files in the working branch
  • git commit -m "comment" : used to commit to repository, -m used to add message
  • git pull origin main : used to pull main branch to make sure it is up to date

JavaScript

  • A variable / var is used to store data in our code.
  • Java variable rules : var "name of variable (shape)" "operator (=)" "data stored ("circle")";
